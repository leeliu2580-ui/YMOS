# 自媒体选题列表 MVP Implementation Plan（修订版）

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建“财经自媒体选题列表 MVP”，实现热点采集、聚类去重、选题生成、审校确认与 Markdown 导出的一条最小闭环。

**Architecture:** 采用“FastAPI 服务 + SQLite + Harness 基础设施 + n8n 调度 + 腾讯云混元 API（主力） + Streamlit 审校页”的最小架构。MVP 仅处理“选题列表”，不包含爆款拆解、长文生成、配图、多平台发布与投研模块，确保先把单主线闭环跑通。

**模型调用策略：**
- MVP 阶段统一使用 **腾讯云混元 API（Coding Plan Pro）** 作为模型服务，用于选题角度生成、标签、初筛等任务。
- 通过统一的 `ModelAdapter` 封装，便于后续替换或增加其他模型（如本地模型作为降级）。
- 若腾讯云 API 不可用，则降级到规则生成的简单角度，不中断流程。

**Tech Stack:** Python、FastAPI、SQLite、SQLAlchemy、Pydantic、httpx、BeautifulSoup4、feedparser、RapidFuzz、scikit-learn（或简单规则聚类）、Streamlit、n8n、Markdown 导出、结构化日志与 trace_id、腾讯云混元 SDK / HTTP API

### MVP 阶段关于模型调用的明确决策
- **主力**：腾讯云混元 API（Coding Plan Pro），用于选题角度生成、标签、摘要等任务。
- **降级**：若腾讯云 API 调用失败（网络、配额超限），则使用规则生成的简单角度，确保流程不中断。
- **可扩展**：通过 `LLMAdapter` 统一封装，后续可无缝接入其他模型（如本地 Qwen、GPT 等）。
- **成本控制**：MVP 阶段限制每天最多调用 100 次腾讯云 API，避免意外超额。

### MVP 的 Harness 前置约束
- 所有 API 响应默认应携带 `trace_id`，用于日志、审校与导出追踪
- 所有关键任务至少定义最小错误枚举，避免把异常文本直接暴露给上层调用方
- 审校动作、导出结果、任务入口都应使用结构化字段，而不是仅靠自然语言描述状态
- Markdown 导出需预留 `task_id / trace_id / generated_at / source_count` 等元数据位
- Streamlit 审校页的动作命名、字段标签与状态值应保持稳定，便于未来被 OpenClaw / Codex 等智能体读取与操作

```python
# apps/media_mvp/services/harness.py
from __future__ import annotations

from datetime import datetime, UTC
from uuid import uuid4


def make_trace_id() -> str:
    return uuid4().hex


def ok(payload: dict | None = None) -> dict:
    body = {"status": "ok", "trace_id": make_trace_id()}
    if payload:
        body.update(payload)
    return body


def envelope(data: dict | None = None, *, status: str = "ok") -> dict:
    body = {"status": status, "trace_id": make_trace_id(), "generated_at": datetime.now(UTC).isoformat()}
    if data:
        body.update(data)
    return body
```

```python
# apps/media_mvp/services/dedupe_cluster.py
from rapidfuzz import fuzz


def cluster_titles(items: list[dict], threshold: float = 0.6) -> list[list[dict]]:
    clusters: list[list[dict]] = []
    for item in items:
        title = item["title"]
        matched = False
        for cluster in clusters:
            score = fuzz.ratio(title, cluster[0]["title"]) / 100
            if score >= threshold:
                cluster.append(item)
                matched = True
                break
        if not matched:
            clusters.append([item])
    return clusters
```

```python
# apps/media_mvp/services/llm_adapter.py
import httpx
from tencentcloud.common import credential
from tencentcloud.hunyuan.v20230901 import hunyuan_client, models
from apps.media_mvp.config import TENCENT_SECRET_ID, TENCENT_SECRET_KEY, TENCENT_MODEL

class LLMAdapter:
    def __init__(self, provider: str = "tencent"):
        self.provider = provider
        if provider == "tencent":
            cred = credential.Credential(TENCENT_SECRET_ID, TENCENT_SECRET_KEY)
            self.client = hunyuan_client.HunyuanClient(cred, "ap-guangzhou")
        else:
            raise ValueError(f"Unsupported provider: {provider}")

    def chat(self, messages: list[dict], **kwargs) -> str:
        if self.provider == "tencent":
            req = models.ChatCompletionsRequest()
            req.Model = TENCENT_MODEL
            req.Messages = messages
            req.Stream = False
            resp = self.client.ChatCompletions(req)
            return resp.Choices[0].Message.Content
        # 其他 provider 可扩展
        raise NotImplementedError

    def generate_angle(self, topic_title: str) -> str:
        """为给定选题生成建议角度"""
        prompt = f"请为以下财经选题提供一个简洁、有吸引力的角度建议（20字以内）：\n{topic_title}\n角度："
        return self.chat([{"Role": "user", "Content": prompt}])
```

```python
# apps/media_mvp/services/topic_score.py
from collections import Counter
from apps.media_mvp.services.llm_adapter import LLMAdapter

_adapter = None

def get_adapter():
    global _adapter
    if _adapter is None:
        _adapter = LLMAdapter(provider="tencent")
    return _adapter

def build_topic_candidates(clusters: list[list[dict]]) -> list[dict]:
    candidates = []
    for cluster in clusters:
        titles = [item["title"] for item in cluster]
        title = Counter(titles).most_common(1)[0][0]
        # 尝试调用模型生成角度，失败则降级为规则角度
        try:
            angle = get_adapter().generate_angle(title)
        except Exception:
            angle = f"围绕“{title}”做财经快评或解读"
        candidates.append(
            {
                "title": title,
                "heat_score": len(cluster),
                "source_count": len({item.get('source', '') for item in cluster}),
                "suggested_angle": angle,
            }
        )
    return sorted(candidates, key=lambda x: x["heat_score"], reverse=True)
```

```python
# apps/media_mvp/services/export_markdown.py
from datetime import datetime, UTC


def render_topics_markdown(candidates: list[dict], *, trace_id: str = "", task_id: str = "") -> str:
    lines = ["# 今日选题列表", ""]
    lines.append(f"> generated_at: {datetime.now(UTC).isoformat()}")
    if trace_id:
        lines.append(f"> trace_id: {trace_id}")
    if task_id:
        lines.append(f"> task_id: {task_id}")
    lines.append("")
    for idx, item in enumerate(candidates, start=1):
        lines.append(f"## {idx}. {item['title']}")
        lines.append(f"- 建议角度：{item['suggested_angle']}")
        lines.append(f"- 来源数：{item.get('source_count', 0)}")
        lines.append("")
    return "\n".join(lines)
```
