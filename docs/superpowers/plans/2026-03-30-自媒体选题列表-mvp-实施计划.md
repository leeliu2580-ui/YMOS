# 自媒体选题列表 MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建“财经自媒体选题列表 MVP”，实现热点采集、聚类去重、选题生成、审校确认与 Markdown 导出的一条最小闭环。

**Architecture:** 采用“FastAPI 服务 + SQLite + n8n 调度 + 可选本地模型初筛 + Streamlit 审校页”的最小架构。MVP 仅处理“选题列表”，不包含爆款拆解、长文生成、配图、多平台发布与投研模块，确保先把单主线闭环跑通。

**OpenClaw 集成决策：** MVP 阶段默认仅对 RSS 和公开静态页面使用 `httpx` / `feedparser` 等轻量抓取方式，以降低依赖和复杂度；若出现动态页面、反爬或登录态站点，再通过封装适配器接入 `web-access` skill。Phase 2 再统一收敛复杂联网能力。

**Tech Stack:** Python、FastAPI、SQLite、SQLAlchemy、Pydantic、httpx、BeautifulSoup4、feedparser、RapidFuzz、scikit-learn（或简单规则聚类）、Streamlit、n8n、Markdown 导出

import feedparser


def parse_rss_items(xml_text: str, source_name: str) -> list[dict]:
    feed = feedparser.parse(xml_text)
    items = []
    for entry in feed.entries:
        items.append(
            {
                "title": entry.get("title", "").strip(),
                "url": entry.get("link", "").strip(),
                "source": source_name,
            }
        )
    return items

# apps/media_mvp/services/dedupe_cluster.py
from rapidfuzz import fuzz


def cluster_titles(items: list[dict], threshold: float = 0.6) -> list[list[dict]]:
    clusters: list[list[dict]] = []
    for item in items:
        title = item["title"]
        matched = False
        for cluster in clusters:
            score = fuzz.ratio(title, cluster[0]["title"]) / 100
            if score >= threshold:
                cluster.append(item)
                matched = True
                break
        if not matched:
            clusters.append([item])
    return clusters
```

# apps/media_mvp/services/topic_score.py
from collections import Counter


def build_topic_candidates(clusters: list[list[dict]]) -> list[dict]:
    candidates = []
    for cluster in clusters:
        titles = [item["title"] for item in cluster]
        title = Counter(titles).most_common(1)[0][0]
        candidates.append(
            {
                "title": title,
                "heat_score": len(cluster),
                "source_count": len({item.get('source', '') for item in cluster}),
                "suggested_angle": f"围绕“{title}”做财经快评或解读",
            }
        )
    return sorted(candidates, key=lambda x: x["heat_score"], reverse=True)
```

    lines = ["# 今日选题列表", ""]
    for idx, item in enumerate(candidates, start=1):
