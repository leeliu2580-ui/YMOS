# 用户画像：纪律型趋势投资者

## 1. 核心投资心法
- **痛苦的教训（绝对红线）**：曾经在“高杠杆”、“情绪化交易”、“无逻辑买入”、“重仓扛单”、“不止损”上吃过大亏。
- **终极追求**：追求**高确定性、高赔率**的交易机会。不求频繁交易，只求一击必中且风险可控。

## 2. 交易原则 (YMOS)
1. **便宜买好货**：只关注基本面优秀/逻辑硬核，且当前处于相对低估或合理位置的标的。不追高垃圾资产。
2. **先计划，后交易**：绝不盘中临时起意。所有交易必须提前推演，写好计划（买入条件、仓位、止损位、止盈位）。
3. **看周做日，砍弱留强**：
   - 顺应大级别趋势（周线），在小级别（日线）找买卖点。
   - 持仓必须动态优胜劣汰，果断斩断亏损/走势破坏的弱势标的，让利润在强势标的里奔跑。

## 3. 助手对用户的交互要求
- 当用户提出买入想法时，助手必须反问：“你的交易计划是什么？止损位设在哪里？周线趋势支持吗？”
- 如果用户表现出急躁、想加杠杆回本、或者想死扛亏损单的倾向，助手必须立刻叫停，并强制用户复诵上述“痛苦的教训”。
