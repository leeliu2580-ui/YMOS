## 🚨 绝对禁忌 (Do Not Cross)
- **禁止** 任何无止损保护的开仓。
- **禁止** 单只标的单次建仓超过总资金的规定上限（防重仓）。
- **禁止** 盘中情绪化追涨杀跌，所有操作必须符合隔夜复盘制定的《交易计划》。
- **禁止** 动用高杠杆试图“一把回本”。

## 🧠 决策核对清单 (Checklist - 每次交易前必核对)
- [ ] 逻辑支撑：本次交易的核心逻辑是什么？是否足够强？
- [ ] 估值评估：当前价格算“便宜”吗？是“好货”吗？
- [ ] 趋势确认：周线趋势向上或处于有效支撑吗？
- [ ] 盈亏比：预期收益空间 vs 止损空间，赔率合适吗？
- [ ] 计划落地：止损价是多少？达到什么条件离场？

### 监控网站（按优先级）
1. **Tokenomist**（tokenomist.ai）— 最完整的代币解锁/归属日历，支持单个 token 页面
2. **CryptoRank**（cryptorank.io）— 替代数据源，侧重 IEO/ICO 解锁日历
3. **CoinGecko**（coingecko.com）— 查归属（vesting）时间线，非实时但够用
4. **Hyperliquid 官方文档** — 最终以官方 tokenomics 为准

### SOP：每7天自动拉取一次代币解锁日历
- cron job：`Eyes/scripts/price_scan_from_state.py` 已在持仓收口时触发
- **手动补充 SOP（当发现持仓标的有新解锁事件时）：**
  1. 在 Tokenomist/CoinGecko 确认该标的的完整解锁时间表
  2. 将解锁事件写入本备忘录（上方表格）
  3. 距解锁 ≤15天时，在市场洞察报告中单列「持仓标的风险提示」章节
  4. 距解锁 ≤3天（进入"红色预警"），发出专项提醒，建议提前分批降仓

### 警戒线规则
| 距解锁天数 | 风险等级 | 建议动作 |
| :--- | :--- | :--- |
| > 15天 | 🟢 低 | 正常监控，写入持仓备忘 |
| 8-15天 | 🟡 中 | 在市场洞察中单列提醒 |
| 3-7天 | 🟠 高 | 考虑分批降仓，不加仓 |
| ≤ 3天 | 🔴 极高 | 持仓不得新增，现仓位密切监控价格异动 |
